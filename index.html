This effect was created with Owl Carousel!

A touch enabled jQuery plugin creates a wonderful responsive slider. It is both professional and visually appealing to the user. At least it was for me! All navigation buttons are functional, but feel free to scroll.

The Why

I wanted to create a professional but fun SPA. I've actually learned so much in creating this project and am proud to say I have new found knowledge, appreciation and love for front end design. The possibilities are endless for what you can do to create a responsive site that dynamically adjusts to the device being used; and that excites me. In this site, I have implemented Bootstrap, jQuery UI plugins, Parallax Scrolling technique, Font Awesome, HTML5shiv and I am very happy with the outcome!

JQuery

jQuery condenses the amount of code used, making what would be somewhat of a task in javaScript, simplified. Using features like PrettyPhoto, a jQuery lightbox clone, supports images, videos, flash, YouTube, iframes and ajax.

Bootstrap Framework

Bootstrap is an amazing framework that efficiently scales the site to fit mobile, desktop, or tablet.When building projects that use sliders, tabs, accordions; these functionalities are added by adding few lines of code. With the customization option you can also choose only certain plugins to keep the file size to a minimum.

Owl Carousel

Owl Carousel is fully responsive with options to drag and click, can implement working buttons on to each slide, browser compatability is optimal, and when coupled with other plugins, the final product is stunning.

Font Awesome

Font Awesome gives you scalable vector icons that can instantly be customized — size, color, drop shadow, and anything that can be done with the power of CSS; if it's not used, a photo would be used for every section that you want to use an icon for and that's just not a good idea!

Button Icons

All buttons on this site are fully functional. The read more buttons for the blog section lead directly to the Zappos blog. The social media buttons at the bottom of the page, all lead to Zappos social media sites. It was simply a little extra fun I was having with the page!

About Me

My name is Sarah Bonilla and I am originally from Louisiana. I have sang on the Las Vegas strip, done massage for hospice, and dabbled in programming before making the passion for it a career goal to achieve in life. I feel like life is a gift to be unwrapped daily and it is meant to be taken as far as possible.

Improving the Front End

When I set out on this journey of code, I knew I wanted to create something beautiful and efficient. I have spent most of my adult life working with the elderly and have noticed a common scrunching of the face when speaking in terms of "the internet." My interest in front end development stems from wanting to create something for everyone; incorporating new and visually appealing projects that are easy to use cross generations, but losing nothing in functionality. I believe it is essential for marketing, the growth and continued success of whatever product or information that is being put out. The way of the future is internet and mobile for everything, so it should appeal to and be easy to use for all.
